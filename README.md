## 📌 Описание

«АЦК-Финансы» — это проект системы бюджетного учёта. Позволяет вести реестр объектов (код, наименование, план), отслеживать кассу, оплату по контрактам, сравнивать периоды и строить статистику. Реализован в двух вариантах:

- Веб-интерфейс — интуитивно понятные формы для ежедневной работы.
- Консольное приложение — текстовое меню для быстрого доступа и автоматизации.

---

## 🎯 Возможности

- Веб-версия  (на скриншотах)

- Аутентификация:  вход по логину/паролю (демо: admin / 12345).

- Работа с объектами:  просмотр, добавление, поиск по коду или имени.

- Платежи: добавление платежа по объекту (сумма, описание, дата).

- Показатели:  просмотр плана, кассы, исполнения по объекту или за период.

- Сравнение периодов:  анализ показателей между двумя датами.

- Общая статистика:  количество объектов, общий план/касса, процент исполнения.

- Навигация:  пункты меню: Объекты, Показатели, По периоду, Сравнение, Поиск, Статистика, Объект, Платёж.

## 🔨Стек технологии

- Фронтенд (веб): HTML%, CSS3, JavaScript (чистый без фреймворков) 
- Консольная версия: C++17 (STL, fstream, iostream)
- Сборка Make, GCC
- Хранение данных JSON 
- Экспорт CSV/XLSX - для открытия в Excel и форматирования отчета

## 💻Инструкция по развертыванию в локальном контуре 


Веб-версия

1. Клонируйте репозиторий:
     git clone https://github.com/yourname/ack-finance.git
   cd ack-finance/web
   2. Откройте index.html в браузере (или запустите локальный сервер).
3. Войдите под демо-учёткой: admin / 12345.

Консольная версия
cd ack-finance/console
g++ -o ask *.cpp

После запуска появится меню — введите номер пункта.
